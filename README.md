# Superstore Sales Analysis Project
## Problem Statement
The goal of this project was to analyze the Superstore Sales dataset (Kaggle) to uncover actionable insights that support business decision-making.
## Tools & Process
• 	Excel → Data cleaning and preparation
• 	Power BI → Analysis, visualization, and dashboard creation
## Key Insights
• 	Total sales: **2.3M** with profits of **286K** across **9994K** orders
• 	**Canon imageCLASS 2200 Advanced Copier** was the top-selling product
• 	Sales grew by **51% between 2014–2017**, with California leading state-level performance
• 	Balanced category contributions across Furniture, Office Supplies, and Technology
## Recommendations
Product Strategy
• 	Double down on Canon imageCLASS 2200 with bundles, loyalty programs, and targeted campaigns
• 	Expand high-performing categories (Technology) with premium upsells and B2B channels
Regional Sales Optimization
• 	Focus on California, New York, and Washington with geo-targeted promotions
• 	Revive low-sales states (e.g., Maine, West Virginia) using localized campaigns
Profitability Enhancement
• 	Review low-margin products and adjust pricing strategies
Customer & Order Insights
• 	Increase average order value through upselling/cross-selling bundles
• 	Segment customers (new vs. repeat) for tailored retention campaigns
